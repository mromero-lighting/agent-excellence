# Agent Excellence — Cooper Lighting Solutions

Agency management portal for Cooper Lighting Solutions. Built with React + Vite.

## Features
- Agency Scorecard (4 segments: Stock, Spec, Industrial, Connected)
- Action Tracker (Last 30 / Current / Next 30 days)
- Training Hub with sequential learning paths
- Progress & monthly comparisons
- Knowledge Base with user-created cards
- Community Q&A Forum
- Role-based access (Admin, VP, RSM, Agency Owner, Staff)

## Dev
```bash
npm install
npm run dev
```

## Deploy
Automatically deployed via Vercel on push to `main`.
